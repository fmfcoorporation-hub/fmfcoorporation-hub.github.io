Bab 3 - Fungsi Logika

Daftar Isi: a. IF Family b. Operator Logika c. Kontrol Logika Tambahan d. Fungsi Definisi & Kustom e. Fungsi Dinamis f. Fungsi Informasi Logika

a. IF Family

IF

Deskripsi: Mengembalikan satu nilai jika kondisi benar, dan nilai lain jika salah.

Sintaks: =IF(logical_test, value_if_true, value_if_false)

AB
180=IF(A1>=60,"Lulus","Gagal")

Hasil: Lulus

IFS

Deskripsi: Mengevaluasi beberapa kondisi secara berurutan.

Sintaks: =IFS(kondisi1, hasil1, kondisi2, hasil2, ...)

IFERROR

Deskripsi: Mengembalikan nilai khusus jika rumus menghasilkan error.

Sintaks: =IFERROR(value, value_if_error)

IFNA

Deskripsi: Mengembalikan nilai khusus jika hasil adalah #N/A.

Sintaks: =IFNA(value, value_if_na)

b. Operator Logika

AND

Deskripsi: Mengembalikan TRUE jika semua argumen benar.

Sintaks: =AND(logical1, [logical2], ...)

OR

Deskripsi: Mengembalikan TRUE jika salah satu argumen benar.

Sintaks: =OR(logical1, [logical2], ...)

NOT

Deskripsi: Membalikkan logika, TRUE menjadi FALSE.

Sintaks: =NOT(logical)

XOR

Deskripsi: TRUE jika jumlah kondisi benar adalah ganjil.

Sintaks: =XOR(logical1, logical2, ...)

c. Kontrol Logika Tambahan

SWITCH

Deskripsi: Memilih dari daftar nilai berdasarkan ekspresi.

Sintaks: =SWITCH(expression, value1, result1, [default])

CHOOSE

Deskripsi: Memilih nilai dari daftar berdasarkan nomor indeks.

Sintaks: =CHOOSE(index, value1, value2, ...)

CHOOSECOLS

Deskripsi: Mengambil kolom tertentu dari array.

Sintaks: =CHOOSECOLS(array, col_num1, [col_num2], ...)

CHOOSEROWS

Deskripsi: Mengambil baris tertentu dari array.

Sintaks: =CHOOSEROWS(array, row_num1, [row_num2], ...)

d. Fungsi Definisi & Kustom

LET

Deskripsi: Mendefinisikan nama untuk hasil perhitungan dalam rumus.

Sintaks: =LET(name1, value1, calculation)

LAMBDA

Deskripsi: Membuat fungsi kustom dengan parameter.

Sintaks: =LAMBDA(parameter, calculation)

BYROW

Deskripsi: Menghitung rumus untuk setiap baris array.

Sintaks: =BYROW(array, LAMBDA(parameter, calculation))

BYCOL

Deskripsi: Menghitung rumus untuk setiap kolom array.

Sintaks: =BYCOL(array, LAMBDA(parameter, calculation))

e. Fungsi Dinamis

FILTER

Deskripsi: Menyaring data sesuai kriteria.

Sintaks: =FILTER(array, include, [if_empty])

UNIQUE

Deskripsi: Menghasilkan daftar nilai unik dari array.

Sintaks: =UNIQUE(array, [by_col], [exactly_once])

SORT

Deskripsi: Mengurutkan array.

Sintaks: =SORT(array, [sort_index], [sort_order], [by_col])

SORTBY

Deskripsi: Mengurutkan array berdasarkan array lain.

Sintaks: =SORTBY(array, by_array1, sort_order1, ...)

f. Fungsi Informasi Logika

TRUE

Deskripsi: Mengembalikan nilai logika TRUE.

FALSE

Deskripsi: Mengembalikan nilai logika FALSE.

ISLOGICAL

Deskripsi: Memeriksa apakah nilai adalah logika (TRUE/FALSE).

Sintaks: =ISLOGICAL(value)
